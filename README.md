# Weatherly

A modern weather application built with Flutter that combines accurate weather information with a clean and intuitive user experience. The app is designed to make checking the weather effortless, providing essential details in a visually appealing interface.

## Overview

Weatherly delivers real-time weather updates, forecasts, and key atmospheric information in a layout that prioritizes readability and ease of use. The application focuses on creating a smooth user experience while maintaining fast performance across devices.

## Features

* Real-time weather information
* Hourly and daily forecasts
* Current temperature and weather conditions
* Humidity, wind speed, and atmospheric details
* Responsive design for different screen sizes
* Clean and modern Flutter-based UI
* Smooth animations and transitions
* Lightweight and fast performance

## Getting Started

### Prerequisites

* Flutter SDK
* Dart SDK
* Android Studio or VS Code
* A valid weather API key

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/weatherly.git
```

2. Navigate to the project directory

```bash
cd weatherly
```

3. Install dependencies

```bash
flutter pub get
```

4. Configure your API key

Create the required configuration file and add your weather API key.

5. Run the application

```bash
flutter run
```

## Design Philosophy

The application was built with a simple goal: weather information should be easy to access and pleasant to read. Every screen and component is designed to reduce clutter while highlighting the information users care about most.
